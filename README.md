## 📖 About the Project

The **NITI IAS Coaching Platform** is a full-stack educational web application developed to streamline the management and delivery of UPSC, IAS, and State PCS coaching services.

The platform provides a centralized system where students can explore coaching programs, read educational blogs, practice Previous Year Questions (PYQs), and access learning resources through a modern and responsive interface.

Alongside the student portal, the application includes a secure administrative dashboard that enables administrators to manage blogs, courses, study materials, PYQs, user profiles, and notifications without directly interacting with the database.

The backend exposes a RESTful API built with Express.js and MongoDB, while the frontend is developed using React and Vite for a fast, responsive user experience. Authentication is implemented using JSON Web Tokens (JWT), and Socket.IO is integrated to support real-time notifications.

The platform follows a modular architecture that separates frontend, backend, business logic, routing, middleware, and database models, making the project easier to maintain and extend.

* Real-time Notifications with Socket.IO

* MongoDB Database Integration

* Modular Project Structure
